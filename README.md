## Overview

**MSDT** is a multi-sensor SLAM benchmark dataset designed for evaluating localization and mapping algorithms in **degraded tunnel environments**.

Unlike many existing SLAM datasets collected in urban, campus, indoor, or mixed environments, MSDT focuses on long tunnel scenes characterized by:

- Repetitive structures
- Weak visual texture
- Limited geometric diversity
- Poor or varying illumination
- GNSS-denied operation
- Long-range drift accumulation risk

The dataset provides synchronized multi-modal sensor data collected using **UAV**, **handheld**, and **ground-vehicle** platforms. It also provides ground-truth trajectories generated by a low-cost tightly coupled **LiDAR–IMU–UWB** framework.

The dataset is intended to support research on:

- LiDAR SLAM
- LiDAR–inertial odometry
- Visual–LiDAR–inertial SLAM
- Multi-sensor fusion
- Degeneracy-aware SLAM
- Robust localization in tunnels and GNSS-denied environments
- Low-cost ground-truth acquisition

## Key Features

- **Dedicated degraded tunnel benchmark**  
  MSDT focuses specifically on long tunnel environments with structural repetition, weak texture, and severe drift risk.

- **Multi-platform acquisition**  
  Data are collected using UAV-based, handheld, and vehicle-mounted platforms.

- **Multi-modal sensor suite**  
  The dataset provides LiDAR, IMU, monocular RGB camera, and UWB measurements.

- **Hardware-synchronized sensor data**  
  The LiDAR, IMU, and camera measurements are hardware-synchronized. UWB measurements are temporally aligned through software-level synchronization.

- **Low-cost ground-truth acquisition**  
  Ground-truth trajectories are generated using a tightly coupled LiDAR–IMU–UWB framework, avoiding expensive surveying systems such as robotic total stations or high-grade INS.

- **Sparse UWB infrastructure**  
  The current implementation requires only a single UWB anchor, enabling low-cost and rapid deployment in tunnel environments.
